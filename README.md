# GNCE Mercury Robotics Team Website

Official website for GNCE Mercury FTC team #26413. This site showcases our team, sponsors, and robot so people can learn more about us.

Visit the website [here](https://gnce-mercury.vercel.app)

## Technologies Used:

**Framework**: Next.js with App Router

**Language**: Typescript

**Styling**: Tailwind CSS

**Deployment**: Vercel

**Package Manage**: npm
